Honeypot

Um honeypot é um recurso computacional com o objetivo de ser sondado,
atacado ou até mesmo comprometido. Geralmente o honeypot é um host
Internet, possuindo um endereço IP público (isto é, um endereço IP roteável
globalmente, que não pertence a nenhuma faixa de endereços
reservados, mas que não hospeda nenhum serviço anunciado publicamente.
Qualquer interação realizada com UM honeypot já pode ser considerada
suspeita, já que é necessária a realização de uma varredura para a
descoberta do endereço IP do honeypot. O sistema deve ser monitorado de
forma discreta, para evitar que um atacante desconfie de estar sendo
observado.
